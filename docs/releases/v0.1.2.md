# WClock v0.1.2

Patch release focused on settings layout, overlay sizing, and photo frame smoothness.

## Changes

- Settings use compact dropdown controls for photo mode, photo order, weather source, location mode, refresh interval, and weather icon style.
- Clock, weather, and five-day forecast overlays can be enabled or hidden independently.
- Clock and weather panel opacity are configured separately.
- Weather forecast toggle and weather opacity are placed on one row.
- Clock panel width is tightened when seconds are enabled.
- Current-weather-only panel is narrower and shorter when the five-day forecast is disabled.
- Photo frame pan speed is configurable.
- Weather text, spacing, and icons are larger for readability on smaller screens.
