# WClock v0.1.1

Patch release after the initial `v0.1.0` publication.

## Changes

- Clock and weather panels are aligned to the same lower edge in landscape mode.
- Clock panel background height is reduced to remove excess empty space.
- Weather detail line spacing is increased for readability.
- Version bumped to `0.1.1` / `versionCode 2`, so it updates over `v0.1.0`.
